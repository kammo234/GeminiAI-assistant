# 🤖 GeminiAI Assistant

GeminiAI Assistant is a **Flask-based AI chatbot** integrated with **Google Gemini API**.  
It provides a beautiful and responsive **chat interface** built with **HTML, CSS, and JavaScript**,  
allowing users to have intelligent, human-like conversations directly in their browser.

---

## 🌟 Features

- 🧠 Powered by **Google Gemini AI API**
- ⚡ Built using **Python (Flask)** backend
- 💬 Interactive chat interface (HTML, CSS, JS)
- 🪶 Smooth UI with real-time message display
- 📁 Chat logs saved for future reference
- 🔒 Environment variables for API key security
